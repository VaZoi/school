HTML/CSS

HTML en CSS zijn de basis programmeertalen voor het maken van een website. Html staat voor Hypertext Markup Language. Dit is een codetaal die gebruikt wordt om structuur en inhoud te geven aan jouw webpagina. Om CSS te kunnen gebruiken is er een HTML-pagina nodig. CSS wordt gebruikt voor de opmaak van de pagina.

Wat heb je nodig om HTML code te kunnen schrijven en runnen

  1. doctype tag + html
  2. html tag
  3. head tag
  4. body tag
